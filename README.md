# TO-DO-LIST
A fully functional and visually appealing To-Do List Web Application that helps users manage tasks, set reminders, track productivity, and stay focused. This project is built using HTML, CSS, and JavaScript with localStorage for data persistence. No backend or database required.

# Features
1. Task Management: Create, view, and mark tasks as complete.
2. Focus Timer: Pomodoro-style timer to improve focus and productivity.
3. Reminders: Set reminders for tasks with time-based alerts.
4. Notes Section: Save important notes and thoughts.
5. Analytics Dashboard: View daily, weekly, and monthly progress charts.
6. Authentication: Signup and Login pages with localStorage-based user validation.

# Project Structure
1. Add-Task.html         # Add and view tasks
2. Analysis.html         # Chart-based task analysis (Chart.js)
3. DashBoard.html        # Navigation hub for all features
4. Focus-Timer.html      # Timer interface (Pomodoro)
5. HomePage.html         # App introduction and navigation
6. LoginPage.html        # User login
7. Notes.html            # Personal note-taking page
8. Set-Reminder.html     # Reminder creation and management
9. SignUp.html            # User signup

# Technologies Used
1. HTML5
2. CSS3 (with gradients, shadows, responsive layout)
3. JavaScript (Vanilla)
4. Chart.js for data visualization
5. LocalStorage API

# Getting Started
1. Clone or download the repository.
2. Open HomePage.html in a web browser.
3. Navigate through the features using the UI.

Note: This app uses localStorage, so your data will persist in the same browser but is not synced across devices.
